# Predictive Analytics Using Historical Data

## Project Overview
This project builds a predictive analytics model using historical sales data to forecast future sales trends.

## Features
- Data Cleaning and Preprocessing
- Feature Engineering
- Linear Regression Model
- Model Evaluation (MAE, MSE, RMSE, R² Score)
- Historical Sales Trend Visualization
- Actual vs Predicted Sales Comparison
- Next 7 Days Sales Forecast

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

## Result

The Linear Regression model achieved an R² Score of approximately **0.99**, demonstrating excellent prediction accuracy on the historical dataset.
